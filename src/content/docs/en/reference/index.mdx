All-contributors is a specification for recognizing contributors to an open-source project in a way that rewards every contribution, not just code.

The basic idea is this:

> Use the project `README.md` (or another prominent public documentation page in the project) to recognize the contributions of members of the project community.

People are giving themselves and their free time to contribute to open source projects in so many ways, so we believe everyone should be praised for their contributions (code or not).
