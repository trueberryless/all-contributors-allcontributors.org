In this reference list you can see all supported contribution types of All Contributors.

## About the Emoji Key

To have a contribution added when using the [Bot](/en/bot) or [CLI](/en/cli), use one or more of the keywords listed in the `Contribution` column [below](#contributions).

:::caution
Contribution keywords must match the types supported by the All Contributors specification and CLI exactly as written below.
:::
